# 🏫 Sistema Integral OGBU - UNMSM

Sistema web desarrollado para la **Oficina General de Bienestar Universitario (OGBU)** de la Universidad Nacional Mayor de San Marcos, orientado a optimizar la gestión y atención de **residentes, postulantes y personal administrativo**.

---

## 📌 Descripción

Este sistema permite centralizar procesos clave de bienestar universitario, facilitando la interacción entre los diferentes actores mediante una plataforma moderna, accesible y eficiente.

El objetivo principal es mejorar la gestión de servicios como residencias, postulaciones, seguimiento y administración interna.

---

## 🎯 Funcionalidades

### 👨‍🎓 Residentes
- Registro e inicio de sesión
- Consulta de estado de residencia
- Solicitud de servicios (alimentación, alojamiento, soporte)
- Reporte de incidencias
- Seguimiento de solicitudes

### 📝 Postulantes
- Registro de postulaciones en línea
- Subida de documentos requeridos
- Validación de requisitos
- Consulta de resultados
- Notificaciones de estado

### 👨‍💼 Personal Administrativo
- Gestión de postulaciones
- Evaluación y validación de documentos
- Administración de residentes
- Generación de reportes
- Panel de control (dashboard)

---

## 🛠️ Tecnologías Utilizadas

- **Backend:** Python (Flask)
- **Frontend:** HTML, CSS, Bootstrap
- **Base de datos:** MySQL / SQLite
- **Control de versiones:** Git & GitHub
